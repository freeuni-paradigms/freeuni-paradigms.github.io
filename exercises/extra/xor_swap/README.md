# Xor Swap

### პირობა:
დაწერეთ xor_swap ფუნქციის იმპლემენტაცია xor(^) ოპერატორის გამოყენებით, რომელიც არგუმენტად მიიღებს ორ int ტიპის ცვლადზე მისამართს და გაუცვლის მათ ადგილებს.

<b>მაგალითი:</b><br>
int x = 2, y = 3;<br>
xor_swap(&x, &y);<br>
ფუნქციის გამოძახების შემდეგ:<br>
y = 2<br>
x = 3
